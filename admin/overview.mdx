The AdVerify admin panel is a Next.js application that gives you full control over the platform. It communicates directly with the `/api/admin/*` server routes using HTTP Basic auth and provides a visual interface for all management tasks.

## Logging in

Open the panel URL in your browser. You are prompted for a username and password that correspond to the `ADMIN_USERNAME` and `ADMIN_PASSWORD` environment variables set on your server. All subsequent requests carry those credentials automatically.

<Note>
The session is stored in memory. Refreshing the page requires you to re-enter credentials unless your browser has saved them.
</Note>

## Dashboard metrics

The dashboard loads two API calls on mount:

- `GET /api/admin/stats` — platform-wide counters
- `GET /api/admin/user-pins/stats` — PIN activity for today and yesterday

### Platform stats

| Metric | Description |
|---|---|
| API Keys | Total number of registered API keys |
| Total Ads | Total ads across all apps |
| Total PINs | All-time PIN records in the database |
| Used PINs | PINs that have been verified at least once |
| Impressions | All-time ad impressions |
| Today Impr. | Impressions recorded today (UTC) |
| Total Clicks | All-time ad clicks |
| Today Clicks | Clicks recorded today (UTC) |
| CTR | Click-through rate: `(total clicks / total impressions) × 100` |

### PIN analytics

| Metric | Description |
|---|---|
| PINs Today | PINs generated on the current day |
| Used Today | PINs verified on the current day |
| PINs Yesterday | PINs generated on the previous day |
| Used Yesterday | PINs verified on the previous day |
| Active PINs | PINs that are verified and not yet expired |
| Expired PINs | PINs that have passed their `expiresAt` timestamp |
