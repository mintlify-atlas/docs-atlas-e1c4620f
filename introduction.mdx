# AdVerify SDK

AdVerify is a complete ad monetization and device verification system for Android apps. It gives app developers a drop-in solution to gate app access behind a monetized PIN verification flow, serve interstitial ads, and track ad performance — all managed from a web-based admin panel.

## What AdVerify does

AdVerify lets you:

1. **Require PIN verification** before users can access your app — forcing them through a monetized verification flow (view ads or articles to receive a PIN)
2. **Serve interstitial ads** inside the app via a dialog-based ad system with zero external dependencies
3. **Track impressions and clicks** on ads through the backend analytics
4. **Manage everything** from a web-based admin panel

## How it works

```
App Launch
    │
    ▼
SDK calls /api/sdk/init with deviceId
    │
    ├── pinEnabled=false OR pinVerified=true
    │       → Fetch and show ads directly
    │
    └── pinEnabled=true AND pinVerified=false
            → Show PIN Verification Dialog
                │
                ├── User clicks "Generate PIN & Verify"
                │       → Server creates shortener link
                │       → Browser opens (user views ads/articles)
                │       → User receives PIN
                │
                └── User enters PIN → /api/sdk/verify-pin
                        │
                        ├── Valid → Dismiss dialog → Show ads
                        └── Invalid → Show error + remaining attempts
```

## Tech stack

| Component | Technology |
|-----------|-----------|
| Android SDK | Java, min API 21, zero dependencies |
| Backend server | Express.js, TypeScript, Prisma ORM |
| Database | PostgreSQL |
| Admin panel | Next.js (App Router), TypeScript |
| Verification flow | Shortener links via paidappstore.com API |
